### 介绍
游戏的中文名为疯狂跳跳跳，以前的名字叫跳一跳，是一个自制python小游戏  
打开版本文件夹后，打开 `main.py` 运行  
有些版本的 `__pycache__` 文件可能没删，这些文件不需要下载
